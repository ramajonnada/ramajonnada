## 🚀 What I'm building

### [Zyra UI](https://zyraui.dev) — Angular Component Library
A modern Angular component library with design tokens, signals-first components, and dark-mode-first theming.

- 📦 `npm install zyra-ng-ui`
- ⚡ Angular 21 + Signals + SSR ready
- 🎨 Token-driven theming — swap one file to rebrand
- 🌙 Dark-mode first, light-mode perfect
- ♿ WCAG 2.1 AA accessible out of the box
